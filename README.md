Output format is hh:mm:ss (including the ":")

----------------------------------------------------------

For example: 

# 08:24:50

* Expansion
* Solderless Breadboard MB-102
* Mini Board
* Jumper Wire x 16
* Matrix Led 8x8
* Battery 9V Cable
* USB Cable
